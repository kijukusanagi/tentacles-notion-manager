# Notion Template

> **Template URL:** [https://tentacles-manager.notion.site/Tentacles-Management-Layer-3276026675c5817f9668eb0c557689fe](https://tentacles-manager.notion.site/Tentacles-Management-Layer-3276026675c5817f9668eb0c557689fe)

Duplicate this template into your Notion workspace to get started.

## What's included

- 8 interconnected databases with full schemas, views, and formulas
- Pre-wired two-way relations between all databases
- Hub pages (OS Layer, Dashboard, Team, OKRs, Partnerships)
- Documentation pages with placeholder values (the agent fills these in during onboarding)
- Empty Project Code enum (the agent populates this during setup)

## How to duplicate

1. Click the template link above
2. Click "Duplicate" in the top-right corner
3. Choose which workspace to duplicate into
4. Wait for duplication to complete (takes ~30 seconds)

## After duplicating

All relations should point to your duplicated databases (not back to the source template). The onboarding agent will verify this automatically during setup.
