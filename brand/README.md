# Tentacles Brand Assets

Pixel-art brand imagery for the Tentacles project. All images feature the Tentacles octopus mascot in an underwater setting.

## Image Index & Recommended Usage

| File | Description | Recommended Use |
|------|-------------|-----------------|
| `01-database-cubes.png` | Octopus organizing glowing data cubes on shelves — databases as physical objects being cataloged | Database documentation, OS Layer System Map page |
| `02-security-guard.png` | Octopus in security cap with ring of keys, guarding a vault door | Permissions docs, security configuration, Agent Permissions page |
| `03-octopus-bold.png` | Bold hero shot — octopus front-facing with intense yellow eyes, powerful pose | Main OS Layer page cover, hero image, GitHub README |
| `04-octopus-soft.png` | Softer hero shot — similar pose but friendlier, more approachable | Teamspace Home cover, onboarding pages, welcome screens |
| `05-octopus-servers.png` | Octopus holding a wrench, working on server racks and equipment | Agent Config page, infrastructure docs, server setup |
| `06-octopus-oscilloscope.png` | Octopus operating an oscilloscope, monitoring signals | Ticket System Interface Spec, monitoring, diagnostics |
| `07-contemplative.png` | Octopus with thought bubble, deep in thought underwater | Agent System Prompt page, AI/thinking content |
| `08-team-crew.png` | Octopus surrounded by sea creature team (starfish, crab, seahorse, jellyfish) all in hard hats | Team pages, Company Dashboard, collaboration content |
| `09-partnership-whale.png` | Octopus and whale meeting at a stone table with coffee and documents | Partnerships page, client meetings, collaboration |
| `10-workflow-factory.png` | Octopus at center of an isometric factory with INPUT and OUTPUT conveyor belts, processing items through a pipeline | Workflow docs, Agent Workflow Templates, ticket processing |

## Style Notes

- **Art style:** Pixel art / retro game aesthetic
- **Color palette:** Deep ocean blues/purples, with neon cyan/magenta/green accents on interactive elements
- **Mascot:** Purple octopus with yellow eyes, orange-pink suckers
- **Setting:** Underwater with coral, ruins, and bioluminescent elements
- **Resolution:** All images are high-res PNGs suitable for web, docs, and Notion covers

## Usage in Notion

These images are designed to be used as **page covers** in the Tentacles Notion template. The recommended mapping above matches each image's theme to the appropriate page.

To set a cover in Notion: Open any page → hover at the top → click "Add cover" → "Upload" → select the image.

Or set covers programmatically using the raw GitHub URLs:

```
https://raw.githubusercontent.com/{YOUR_USERNAME}/tentacles-notion-manager/main/brand/01-database-cubes.png
```
